日记（2022-7-20）

游鱼

也有好一阵子没记日记了。这阵子不光日记，连博客也很少打理。可就在没有打理的这期间，访客都达到 5000 人了。明明只是去年学 Java 的时候分享过几次，不知道他们是从哪看到的链接。

这阵子我也没闲着。老房子这边加装了空调，夏天总算是能待了。然后又去补办了身份证，虽然还没送到，但是这是我 6 年来第一次出门，第一次和别人打交道。总之每一步都很难，但开始动了，所有事情又都比预想的顺利。

之前新学了 Go 语言。Go 语言我真是非常喜欢。Java 胜在功能强大，丰富的类库能满足各种需求，因此在周赛时候我还是倾向于使用 Java。但除此之外，不论是语法，还是各类机制，Go 语言几乎是完胜。毕竟 Go 语言是比 Java 更新的语言，确实能感觉到时代的进步。因此，除了周赛外的场合，我都在用 Go 语言了。

虽然很喜欢 Go 语言，对 Rust 也很有兴趣，另外 Java 也有高级部分等着我，但是这些都被我搁置了。毕竟眼下最紧要的还是自考。如果不能拿到学历，学得再多也没有用。因此，最近的重点还是学习软件工程。

软件工程偏向于理论，只啃书本给我带来了很大的困扰。近年来我学东西都倾向于记笔记，而软件工程的各种流图也很难用 mermaid 语法表示。感谢互联网时代，我在网上找到了一份北大的软件工程的教课视频。让人惊讶的是我这本自考教材居然和他们讲课的内容能够配套，真是天助我也。有了这些帮助，预计 8 月中旬前就能把这一科学完一遍了。其余的如 Java、C++、数据库 这些我早已学过，想必不用再花大力气。

家里的小猫又生了，上回是母猫占了绝大多数，这回生了 3 只公猫。这几天也睁开眼睛了。他们的姐姐站在一边，不跑也不靠近，不知道心里怎么看待她的弟弟们。上回生完没几天就又怀了，这回我们得吸取教训。

前几天给鱼缸里投喂时，发现一条鱼病怏怏的，潜在水底，背鳍收拢，也不动弹。她看见我喂食，起先还用力向上游了游，可很快没了力气，又沉入了水底。我只好尝试着给水里投放了一枚 维生素 C 泡腾片，希望她自己能好起来。可是事与愿违。第二天我又来看她，见她的鳍好像都破了，嘴角也有淤血。她的同类居然趁她虚弱攻击她！我赶紧把她捞出来放在另一个鱼缸里，让她和河蚌爷爷和另外两条人畜无害的小家伙待在一起。这回，哪怕是她努力呼吸的时候有鱼食进到嘴里，她都不会向下吞咽了。我们都以为她要不行了。我只好投下 VC 和一点鱼食，让她听天由命吧。

昨再来看望她，看见缸里的水变浑浊了，水面上浮着很多泡沫。刚要难过，却见她从混沌中游过。背鳍也展开了，动作也敏捷了，水里的身姿也不是像先前那样有气无力的随波逐流了。我为她换了水。或许是自己以往也处在这样虚弱的状态里挣扎过，即使是对一条鱼也似乎能感同身受。以前的人们写诗去记录花鸟鱼虫，多少也是出于这样的心态吧。

一场大病下来，那条小鱼这会儿有些怕生。我一靠近鱼缸，她就吓得又潜回水底了。不过我假装走开，从门后偷看的时候，看到她正努力地吃饭。能好好吃饭，就能活下去，不是吗？想到这里，我也给自己点了一份外卖。之前我也对自己说过，要好好吃饭的。

希望她明天能好起来。
