符号表（运算符）

符号 含义 符号 含义
\(p⋀ q\) \(p\) 且 \(q\) \(p⋁ q\) \(p\) 或 \(q\)
\(p→ q\) \(p\) 是 \(q\) 的充分条件 \(p↔ q\) \(p\) 是 \(q\) 的充要条件
\(p⇔ q\) 命题公式 \(p\) 等价于 \(q\) \(p⇒ q\) 命题公式 \(p\) 能推理出 \(q\)
\(∀ a\) 任意 \(a\) \(∃ a\) 存在 \(a\)
\(a\in A\) 元素 \(a\) 属于集合 \(A\) \(A\subseteq B\) \(A\) 是 \(B\) 的子集
\(A\subset B\) \(A\) 是 \(B\) 的真子集 \(\emptyset\) 空集
\(A\cup B\) \(A\) 与 \(B\) 的并集 \(A\cap B\) \(A\) 与 \(B\) 的交集
\(|A|\) \(A\) 中元素的个数（基数） \(A-B\) 属于 \(A\) 但不属于 \(B\) 的元素集合
\(A\oplus B\) \((A\cup B)-(A\cap B)\) \(\tilde{}A\) \(E-A\)（\(E\) 是全集）
\({\bigcup}{\mathcal A}\) 集族 \(\mathcal A\) 所有元素的并集 \({\bigcap}{\mathcal A}\) 集族 \(\mathcal A\) 所有元素的交集
\(⟨ a,b⟩\) 有序对，即：\(\{\{a\},\{a,b\}\}\) \(⟨ a,b,c⟩\) 有序三元组，即：\(⟨ ⟨ a,b⟩, c⟩\)
\(A\times B\) 笛卡尔积。\(A\) 元素与 \(B\) 元素形成有序对的集合 \(A^n\) n 维笛卡尔积
\(F\circ G\) 二元关系 \(F,G\) 的逆序合成 \(F\restriction A\) 限制：\(\{⟨ x,y⟩|xFy⋀ x\in A\}\)
\(F[A]\) 象：\(\operatorname{ran}(F\restriction A)\) \(F^{-1}\) 二元关系 \(F\) 的逆
\((M(R))^T\) 矩阵 \(M(R)\) 的转置。即：\(M(R^{-1})\) \(M(R_2)• M(R_1)\) 矩阵的逻辑乘，即：\(M(R_1\circ R_2)\)
\([x]\) \(x\) 的等价类 \(A/R\) \(A\) 的商集。\(A\) 中元素的等价类的集合
\(\preccurlyeq\) 偏序关系：自反、反对称、传递的关系。 \(\prec\) 拟序关系：反自反、传递的关系
\(F:A→ B\) \(F\) 是 \(A\) 到 \(B\) 的偏函数 \(F:A\shortparallel\!\!\!→ B\) \(F\) 是 \(A\) 到 \(B\) 的真偏函数
\(F:A\shortmid\!\!\!→ B\) \(F\) 是 \(A\) 到 \(B\) 的全函数 \(f(A')\) \(A'\) 的象
\(f^{-1}(B')\) \(B'\) 的原像 \(A^+\) 集合 \(A\) 的后继。即：\(A\cup\{A\}\)
\(A\approx B\) \(A\) 与 \(B\) 等势（存在双射） \(A\preccurlyeq• B\) \(A\) 较 \(B\) 优势（存在单射）
\(B\succcurlyeq• A\) \(B\) 较 \(A\) 劣势 \(A\prec• B\) \(A\) 较 \(B\) 绝对优势
\(B\succ• A\) \(B\) 较 \(A\) 绝对劣势 \(A\&B\) 无序积。\(A\) 元素与 \(B\) 元素各一个组成的集合的集合
\((x,y)\) 在无序积中表示 \(\{x,y\}\) \(G_1\cong G_2\) 图 \(G_1,G_2\) 同构
\(G'\subseteq G\) \(G'\) 是 \(G\) 的子图 \(G'\subset G\) \(G'\) 是 \(G\) 的真子图
\(G[V_1]\) 由点集 \(V_1\) 导出的子图 \(G[E_1]\) 由边集 \(E_1\) 导出的子图
\(\overline G\) 图 \(G\) 的补图 \(G-e\) 图 \(G\) 删除边 \(e\) 的图
\(G-v\) 图 \(G\) 删除顶点 \(v\)（及关联边）的图 \(G{\setminus}e\) 图 \(G\) 收缩 \(e\) 边（删边，合并两侧顶点）
\(G\cup(u,v)\) 图 \(G\) 增加新边 \(u\sim v\) 点 \(u,v\) 连通
\(G[v_i]\) 点 \(v_i\) 所在的连通分支 \(u→ v\) 顶点 \(u\) 可达顶点 \(v\)
\(u↔ v\) 顶点 \(u,v\) 双向可达
