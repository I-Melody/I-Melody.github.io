日记（2022-5-6）

本文最后更新于：2022年7月20日 凌晨

早安，晚安

好久不上网，心情平静了很多。稍微窥探了一下，发现还是群魔乱舞，哪怕只是看着都让人心情烦躁。还是继续断网吧。

这几天本来挺自律的，作息时间都安排得井井有条。早上起床先睡个懒觉——懒床可是人生最大的幸福之一，趁着现在还有机会，得好好把握。总之偷一会懒，然后起床学 Java，更新笔记。做完以后就锻炼身体。锻炼完就累了，坐下来练字，回复一点体力。练完字就到了 C++ 的时间了，学完 C++ 就差不多该吃晚饭了。吃完饭可以稍微玩一玩，然后有时间的话就练一练画画。练完就该上床睡觉了，睡觉前读一会书，为明天的 Java 复习整理做点准备。

总之本来应该是蛮顺利的，可到了晚上放下书睡觉的时候却出了意外。睡不着。

从高一到现在，快十年了，还在失眠，真烦人。索性刷刷手机。虽然没 wifi，流量还是有一点的。

结果越看越烦燥。网上的人啥大事小事都只会扣帽子然后阴阳怪气，要么就是在卖货。就不能说点人类的语言吗？罢了，刷刷淘宝吧。看什么都想买，好在下单之前理智回来了。看来也不能刷淘宝。

辗转反侧到 3 点，不小心打开了饿了么，看着吃的突然饿了。这下好，彻底睡不着了。

我这边夜间外卖其实也就三类：烤肉拌饭、烤串、韩式炸鸡

烤串和炸鸡贵且不论，配送费也高。我好歹在家里是做过饭的，看着标价感觉那些商家在侮辱我。好像在说：“喂白痴，快把钱给我，然后拿这几块脂渣堵上自己的嘴”。烤肉拌饭倒是便宜，可想想那些奸商的嘴脸，总有种他们是不是在卖厨余垃圾的感觉。

就这样磨磨蹭蹭的，到了白天。6 点的时候，多数外卖都开业了。这时候的选择就多种多样了。豆腐脑、包子、火烧、锅贴、煎饼果子、卤肉饭。实惠又卫生。

再过一会，各种炒菜、汉堡什么的也都要开始了。趁着还早，点了几个包子，不一会就送到了。

以前包子总爱吃肉馅的，现在倒是渐渐喜欢起素的来了。出于习惯点了一个肉馅的，吃了一半竟觉得腻了。以往多少饭都能吃得下，现在胃也小了。人常说上年纪的人饭量会减少，我岂能在这个岁数就开始服老？减肥什么的等会再说，先努努力吃多几口吧。可肚子不争气地饱了。真难过。

以前熬一晚上，总会感慨感慨，什么看着太阳落下又爬上来，什么家人睡下又醒来之类的。现在倒也是习以为常了。

以前总爱喝凉水，甚至对着水龙头喝生水。现在竟然也在学人烧水泡茶了。甚至有一种往水里泡枸杞的冲动了。是啊，经历了这么多，可得好好活着呀。

以前早上为了睡懒觉从来不吃早饭的，现在是为了省钱不吃午饭。哈哈，这点倒没怎么变嘛。可惜就算少吃一顿，该发胖还是发胖。

总之，又熬夜了。看来明天（今天）的计划是实现不了了呢。半困半醒的状态也不可能做什么事了。真该死，又浪费了一天。时间又少了一天。早晨的空气也没有想象中的好。要我说还是下午的最好，凉爽又有风。早晨的就差点意思，再干净也和我没什么关系。

窗外能听到公交车驶过的声音。其他车辆都是车轮滚动的声音，唯独公交车的声音里还混杂着快散架的颠簸感，以及发动机有气无力的沉重感。想起高中时候，背着大大的书包，每周末回家一次。周六回家前可以在外面拖拖拉拉多呆一会，直到坐上晚上的公交车。那时人也少，我总爱坐在最靠前的位置上，把书包抱在怀里，倚靠着前边的黄色栏杆，看窗外景色穿梭。万家灯火，车水马龙，就像梦一样美好。回到家里好好睡一觉，醒来时候一般都到第二天中午了。稍微收拾收拾包，就得准备下午返校了。上学的路就没那么幸福了。先是祈祷公交车不要晚点，还要祈祷下一辆车空一点，别让我被晚高峰的人群挤死，最后还要祈祷路上不堵车，不然走得再早还是得迟到。

车流的声音，附带上一点点人类活动的声音，真是绝妙的助眠白噪音。如果有微风拂过，树叶沙沙作响就好了。如果有倾盆大雨，一部分落在地上，一部分落在窗玻璃上，地上不平整的地方积了水，车来车往间不经意溅起水花，那就更好了。可是既没有风，也没有雨。

……疲惫感，但还没到极限。按照往常的失眠经验，疲惫感先是会逐渐减少，感觉似乎有点清醒了，然后再到某个时刻，身体突然变得虚弱。那个时候就可以睡觉了。现在还没到那个时候。真慢啊。

好像看到一只蜻蜓飞过，原来这么高的地方也有蜻蜓吗？小时候随父亲上山捉过蜻蜓玩，捏住它两个翅膀它就跑不了了。从那时候到现在，不知道多少年没有上山玩，多少年没有见过蜻蜓了。

不知怎的回想起了在路口顶着大太阳，拿着一支冰工厂或者碎冰冰在吃的自己。在我懵懵懂懂的时候，先行者在我看不到的领域内正越走越远。他们推动这个世界转的越来越快，而我到了今天才跌跌撞撞地发觉世界的变化，踉踉跄跄地开始想追赶他们的脚步。

就像现在，别人互道早安时，我还在纠结于能不能晚安。

有时想，明明已经浪费了那么多年了，还在乎多这一天吗？但是，恰恰是已经浪费了太多时间，已经不多的剩余才更显得珍贵。

八宝粥剩了点底，想撕开包装舔干净，最后忍住了。

别写日记了吧，回床上碰碰运气算了。以往的日记都写不了这么长，反倒是失眠了迷迷瞪瞪的还能胡扯。明明上学的时候最头疼的就是凑 800 字，还自作聪明地算着让每段多出一两个字以便多占一行。现在越写越多，感觉自己像个小丑。
