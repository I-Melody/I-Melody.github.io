日记（2022-5-5）

本文最后更新于：2022年7月14日 晚上

环

今天是个值得纪念的日子。终于把《健身环大冒险》打通关了。

最后一关比预想的难，本来看着桌子上热腾腾散发香味的 “奖励餐”，还在想这 BOSS 怎么这么弱。结果居然有更难的二阶段。打完以后吃的东西都凉透了。不过运动花费的汗水和努力补足了食物因为时间而缺失的风味，最后还是吃到了比以往都更好吃的一餐。

……如果外卖商家的良心和以前一样，给的量和以前一样多，那就更完美了！

学 Java 的时候停过一段时间，其余时候也经常偷懒。不过还是坚持下来了。自以为也就坚持了半年左右，今天回顾运动时间的时候才惊觉，已经是坚持了一年了。

和一年前相比，不知道体重是不是减轻了，但是身体健康多了，体力也好了很多。各方面都比以前好了。心理上自不必说，是好得多了。

一个目标达成以后，又不知道下一个目标在哪里了。好像兜兜转转，又走回到了起点前。不过，这回我不再迷茫了。因为我知道：事物的发展都是螺旋上升的。我并不是回到起点，而是到达了起点的上方。这是初中政治课本上的一句话。简单的句子，放在不同情境下却能品出不同的道理。或许这就是所谓 “大道至简” 吧。虽然我以前文科相对弱一些，但那段时候正好是政治课代表。其实很多知识都是先学会，后发现的。所以现在回想起来，真觉得学校的课本里全是宝藏啊。

和一年前相比，我各方各面都不一样了。

《健身环》最后，灵环告诉我，每个人都有弱点，但是只要向着一个方向坚持努力，就一定能看到不一样的自己。

这句话我以前在其他地方多次听到、看到过。在无数次擦肩而过后，它终于在一次相遇中，改变了我。

韩顺平老师在他的 Java 入门教程里讲过一个卖油翁的故事。他常说 ”我亦无他，唯手熟尔“。

习主席曾经在讲话里强调了一种 ”老黄牛“ 的精神，虽然走得很慢，但稳稳当当地不停歇，总能走到很远。

”有的人因为看到而相信，有的人因相信而看到“，马云走到今天的位置，他身上也有很多值得学习的地方。

刻晴也常说：“心有所向，日复一日，必有精进”。她也是我最喜欢的角色之一。

有一天我突然想到，提瓦特大陆究竟是什么在吸引着我？为什么我会喜欢一些虚拟的人物。刻晴、胡桃、甘雨……只是因为她们外形好看吗？

那一刻，我突然明白了。我知道自己想要的是什么了。

我想到胡桃，她很小的时候爷爷就过世了，她只能一个人挑起往生堂的重担。这份职业让她见证了无数的分别，但她总是那样乐观、开朗。

甘雨她把璃月的一切责任抗在肩上，在漫长的岁月中始终坚定，即使岩王帝君离去也不曾动摇。与人类社会的隔阂想必使她孤独吧，但她没有被孤独击垮。她那样柔弱，却成为了所有人的依靠。

命运对申鹤也不公平。她从小被父母抛弃，孤身一人对抗魔兽。她长久以来远离尘世，再入社会时已是格格不入。但她还是回到了璃月，还收获了友谊。我深知这需要的是极大的勇气。

人类很难体会仙人的情感，也很难想象千年的时光。魈在漫长的岁月中一直在和自己的 “心魔” 抗争，从未屈服。即使同伴都已离去，信念也没有丝毫动摇。

刻晴有很多优点。她有智慧，有勇气，也有能力，她想到什么就果断去做，从不会瞻前顾后。她从不迷茫，从不退缩，从不逃避……她是我想成为的样子。

我的心有那样多的缺点。我失去了很多，内心总是那样悲凉，我想要想胡桃一样达观。我总是感到孤独，我也想像甘雨一样坚强。过去的几年我把自己锁在房间里，逐渐对社会感到胆怯，我也想象申鹤那样勇敢地改变自己。我在逃避中不断地失去，不断地错过，我害怕命运，我也想像刻晴那样，一往无前。

那一天，我明白了，我向往的不是一个虚拟的魔法世界，而是一段又一段美丽的人生。我想要像其中所有人那样，成为自己命运的勇者。那一天，虽然有诸多不舍，我还是删掉了所有游戏、社交平台，我终于下定决心，改变自己。

心有所向，日复一日，必有精进。过去一段时间我的经历，也证明了这一点。
